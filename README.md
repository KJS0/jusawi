# 📸 주.사.위 (주저하는 사진가를 위하여)

## 🎯 프로젝트 개요
사진가들의 촬영 습관을 분석하고 개선할 수 있도록 돕는 EXIF 데이터 기반 분석 도구입니다.

### 📋 캡스톤 디자인 목표

#### 1. 배경 및 필요성
- **EXIF 데이터 활용 부족**: 사진 촬영 시 생성되는 풍부한 메타데이터를 직관적으로 활용할 수 있는 도구 부족
- **촬영 습관 분석 도구 필요**: 사진가들이 자신의 촬영 패턴을 객관적으로 분석할 수 있는 솔루션 부재
- **데이터 시각화 필요성**: 복잡한 촬영 데이터를 이해하기 쉬운 형태로 제공하는 도구 필요

#### 2. 주요 수행 내용
1. **EXIF 데이터 추출 및 분석**
   - 사진에서 촬영 정보(조리개, 셔터 속도, ISO, 초점거리 등) 자동 추출
   - 다중 이미지 일괄 처리 및 통계 분석

2. **촬영 습관 분석 및 시각화**
   - 주로 사용하는 카메라 설정 패턴 분석
   - 촬영 데이터 히스토그램 및 분포도 생성
   - 시간대별, 설정별 촬영 빈도 분석

3. **사용자 맞춤 피드백 제공**
   - 분석된 데이터 기반 촬영 습관 개선 제안
   - 최적 설정 추천 기능
   - 촬영 스타일 객관적 평가

#### 3. 최종 목표
- **정량적 성과물**: EXIF 데이터 분석률 95% 이상, 처리 속도 1초당 10장 이상
- **기능적 목표**: 촬영 습관 분석, 데이터 시각화, 자동 추천 기능 구현
- **사용자 경험**: 직관적 UI/UX를 통한 쉬운 데이터 접근성 제공

## 🚀 주요 기능

### 📊 데이터 분석 기능
- **EXIF 데이터 추출**: 카메라 모델, 렌즈, 촬영 설정 등 자동 추출
- **통계 분석**: 조리개, ISO, 셔터 속도 사용 빈도 분석
- **촬영 패턴 분석**: 시간대별, 장소별 촬영 습관 분석

### 📈 시각화 기능
- **히스토그램**: 설정값별 사용 빈도 시각화
- **분포도**: 촬영 설정 조합 패턴 분석
- **트렌드 차트**: 시간에 따른 촬영 습관 변화 추적

### 🎯 추천 시스템
- **설정 추천**: 과거 데이터 기반 최적 촬영 설정 제안
- **습관 개선**: 촬영 패턴 분석을 통한 개선점 제시
- **스타일 분석**: 개인 촬영 스타일 객관적 평가
