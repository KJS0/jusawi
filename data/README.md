# 📁 샘플 데이터 폴더

이 폴더는 EXIF 분석 도구의 테스트 및 데모를 위한 샘플 이미지 파일들을 저장하는 곳입니다.

## 📋 사용법

1. **샘플 이미지 추가**: EXIF 데이터가 포함된 JPEG 또는 TIFF 파일을 이 폴더에 복사
2. **프로그램에서 테스트**: 메인 애플리케이션에서 "폴더 열기"로 이 폴더 선택
3. **분석 실행**: "EXIF 분석 시작" 버튼으로 샘플 데이터 분석

## 🎯 권장 샘플 데이터

효과적인 테스트를 위해 다음과 같은 다양한 조건의 이미지를 포함하는 것을 권장합니다:

### 📸 촬영 설정 다양성
- **조리개**: f/1.4, f/2.8, f/5.6, f/8, f/11 등 다양한 값
- **ISO**: 100, 400, 800, 1600, 3200 등 다양한 감도
- **초점거리**: 24mm, 50mm, 85mm, 200mm 등 다양한 렌즈

### 🕐 촬영 시간
- 아침, 오후, 저녁, 야간 등 다양한 시간대
- 실내, 실외 등 다양한 환경

### 📷 카메라 및 렌즈
- 다양한 카메라 모델 (Canon, Nikon, Sony 등)
- 다양한 렌즈 (단렌즈, 줌렌즈, 매크로 등)

## 📊 분석 결과 예시

샘플 데이터 분석을 통해 다음과 같은 결과를 확인할 수 있습니다:

- 가장 많이 사용하는 촬영 설정
- 촬영 스타일 분류 (인물, 풍경, 스포츠 등)
- 시간대별 촬영 패턴
- 개인화된 촬영 개선 제안

## ⚠️ 주의사항

- **개인정보 보호**: 개인 사진을 샘플로 사용할 때는 개인정보가 포함되지 않도록 주의
- **저작권**: 저작권이 있는 이미지는 개인 테스트 목적으로만 사용
- **파일 크기**: 대용량 파일은 분석 시간이 오래 걸릴 수 있음

## 📝 파일 형식

지원하는 이미지 파일 형식:
- **JPEG** (*.jpg, *.jpeg) - 권장
- **TIFF** (*.tiff, *.tif)
- **PNG** (*.png) - EXIF 데이터가 있는 경우만

---

> 💡 **팁**: 최소 10장 이상의 다양한 이미지를 사용하면 더 정확한 분석 결과를 얻을 수 있습니다.
